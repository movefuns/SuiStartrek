# Sui dApp TicTacToe Game
domain: https://sui-startrek-euraxluo.vercel.app/

## Starting your dApp
To install dependencies you can run

```bash
pnpm install
```

To start your dApp in development mode run

```bash
pnpm dev
```

## Building

To build your app for deployment you can run

```bash
pnpm build
```
